# Python Installation 

### Update and upgrade package index
```
$    sudo apt-get update
$    sudo apt-get -y upgrade
```

### Check if Python already installed
```
$ python3 -V
```

### Install Python
```
$ sudo apt-get install -y python3-pip
```

### Install Python packages using PIP
```
$ pip3 install package_name
```
